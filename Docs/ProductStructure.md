DREAMWEAVER – PRODUCT STRUCTURE (MVP)

1. Product Type
DreamWeaver is a creative studio interface where users collaborate with AI agents to build structured creative universes.

The experience is dashboard-based, not chat-only.

2. Core User Flow

Step 1: Idea Input
User enters a single creative idea (e.g., "A pirate discovers an ancient AI under the ocean").

Step 2: Creative Director Activation
The system analyzes the idea and asks 3–5 guiding questions to shape direction.

Step 3: Creative Brief Creation
A structured creative brief is generated:
- Theme
- Tone
- Audience
- Visual style
- Core conflict

Step 4: AI Studio Execution
Multiple AI agents begin working in parallel:
- Story Agent → narrative
- World Agent → environment and lore
- Character Agent → roles and personalities
- Visual Agent → image concepts and prompts

Step 5: Live Studio Dashboard
User sees AI agents working in real time with updates:
- "World Agent: Designing underwater ruins..."
- "Story Agent: Writing Act 1..."

Step 6: Universe Output
Final structured output:
- Story
- Characters
- World description
- Scene breakdown
- Visual concepts

Step 7: Save Project
User can revisit, refine, or expand the universe.

3. UI Structure (Key Screens)

- Landing Page (simple idea input)
- Creative Interview Screen (AI asks questions)
- Studio Dashboard (main workspace)
- Universe View (final output)
- Project Library (saved worlds)

4. Core Experience Principle

The system must feel like a creative studio, not a chatbot.

AI is represented as working collaborators, not a single responder.

5. MVP Scope (Must Have)

- Single user flow from idea → universe
- 3–4 AI agents
- Basic dashboard UI
- Structured output (story + world + characters)
- Project saving

6. Out of Scope (for hackathon)

- Multiplayer collaboration
- Monetization
- Marketplace
- Advanced animation or game engine integration
- Complex 3D world rendering

7. Key Differentiator

Instead of generating content in isolation, DreamWeaver simulates a creative team working together in real time to build a coherent universe from a single idea.
