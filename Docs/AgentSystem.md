# DREAMWEAVER – AI AGENT SYSTEM

## 1. Overview

DreamWeaver is powered by a coordinated system of specialized AI agents that simulate a creative studio team.

Each agent has a defined role, responsibility, and perspective. Together, they transform a single idea into a structured creative universe.

The system is designed to behave like a collaborative studio, not a single chatbot.

---

## 2. Core Design Principle

> AI should behave like a team of specialists, not a single generalist model.

Each agent contributes a different creative layer, and all outputs are merged into a unified creative vision.

---

## 3. AI Agents

### 3.1 Creative Director

**Role:**
Guides the entire creative process.

**Responsibilities:**

- Asks clarifying questions to refine user idea
- Defines creative direction
- Ensures consistency across all outputs

**Output:**
Creative brief (tone, theme, audience, style)

---

### 3.2 Story Agent

**Role:**
Builds narrative structure.

**Responsibilities:**

- Writes plot outline
- Defines story arc (beginning, middle, end)
- Creates scenes and conflict

**Output:**
Structured story framework

---

### 3.3 World Agent

**Role:**
Builds the universe.

**Responsibilities:**

- Defines geography and environment
- Creates lore and history
- Establishes rules of the world

**Output:**
World-building document

---

### 3.4 Character Agent

**Role:**
Designs characters.

**Responsibilities:**

- Creates protagonists and antagonists
- Defines motivations and relationships
- Ensures character consistency with story and world

**Output:**
Character profiles

---

### 3.5 Visual Agent

**Role:**
Transforms ideas into visual direction.

**Responsibilities:**

- Generates visual descriptions
- Produces image prompts for generation models
- Maintains visual consistency across universe

**Output:**
Art direction + image prompts

---

### 3.6 Critic Agent

**Role:**
Quality control and consistency checker.

**Responsibilities:**

- Detects contradictions
- Evaluates narrative strength
- Suggests improvements
- Ensures tone consistency

**Output:**
Feedback and revision requests

---

## 4. Workflow Between Agents

1. Creative Director receives user idea
2. Creative Director produces creative brief
3. Brief is passed to all agents
4. Agents generate outputs in parallel
5. Critic Agent evaluates combined output
6. System revises if necessary
7. Final universe is assembled

---

## 5. Shared Creative Memory

All agents read and write to a shared context called:

**Creative Memory**

It includes:

- Tone
- Theme
- World rules
- Character relationships
- Visual style

This ensures consistency across all outputs.

---

## 6. Key System Behavior

- Agents do not operate independently in isolation
- Every output must align with Creative Memory
- Critic Agent has authority to request revisions
- Creative Director maintains overall coherence

---

## 7. Key Differentiator

Instead of generating a single response, DreamWeaver simulates a coordinated creative team that collaboratively builds a universe from a single idea.
